Life Tracker

The Life Tracker App is an application that helps users track and manage their health and wellness activities. It allows users to add activities of choice for instance sleep, running and meditation.For each activity, you specify its measure of success. An activity like sleep could be quantified by number of hours while running could be quantified by number of calories burned or the distance. Each time a user performs an activity, he/she logs it in the application. The landing page of the app presents a summary of these activities and offers visual insight of the individual's progress.

Student Store

The student store is a simple e-commerce website that is tailored for college students. Here, users can browse through available merchandise like stationery, snacks and sporting gear. The website supports a full-fledged shopping experience, allowing users to select an item of choice, specify its quantity and add it to a shopping cart. Upon checkout, a receipt is printed as a summary of the order, taxes and fees.

Light & Sound Game

Build using CSS, HTML and Vanilla Javascript,The Light and Sound Memory game is a simple and interactive web based game. The player keeps track of the autogerated patterns and tries to replicate them by following along with the mouse. There is a winning and losing condition determined by the countdown time and the accuracy of matching the patterns.
